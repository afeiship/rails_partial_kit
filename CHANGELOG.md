## [0.1.0] - 2026-02-20

### Added
- Component registry system for registering UI components
- Component renderer with validation support
- Helper method `component` for rendering components in views
- Support for default values in component configuration
- Support for validation via Hash rules or Proc validators
- Auto-loading of component definitions from `app/components` directory
- Example components: Card, Button, Alert
- Demo application with examples
